# MiniDAQ

## Overview

MiniDAQ is a project relating to creating and testing an ESP32-based modular data acquisition and telemetry system inspired by aerospace flight computers and motorsport telemetry.

The objective is to design a professional embedded system capable of: 
- Acquiring sensor data
- Logging data to local storage
- Transmitting telemetry to ground station
- Visualizing data
- Providing a modular firmware architecture

  ##Current Hardware
  - ESP32
  - MPU6050
  - microSD module (planned)
